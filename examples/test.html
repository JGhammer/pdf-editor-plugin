此页面用于验证 PDF Editor Plugin 的核心功能是否正常工作。 点击下方按钮进行各项功能测试。

📖 快速使用指南

React 项目中使用：

import { PdfEditorReact } from 'pdf-editor-plugin';

function App() {
  return (
    <PdfEditorReact 
      placeholder="请上传PDF文件..."
      onContentChange={(data) => console.log(data)}
    />
  );
}

Vue 3 项目中使用：

import PdfEditor from 'pdf-editor-plugin/src/components/vue/PdfEditor.vue';

export default {
  components: { PdfEditor },
  template: '<PdfEditor @content-change="handleChange" />'
}

Vue 2 项目中使用：

import PdfEditor from 'pdf-editor-plugin/src/components/vue/PdfEditorVue2.vue';

export default {
  components: { PdfEditor },
