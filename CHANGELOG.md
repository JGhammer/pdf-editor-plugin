# PDF Editor Plugin - 更新日志

## v1.0.0 (2026-04-15)

### ✨ 新功能
- 🎉 首次发布
- 📄 PDF 文件上传与解析功能
- ✏️ 富文本编辑器（基于 Quill.js）
- 🖼️ 图片替换功能
- 💧 水印检测与去除
- 📤 PDF 导出功能
- 🎨 现代化深色主题 UI

### 🎯 框架支持
- ✅ React 18+ 支持
- ✅ Vue 3 支持（Composition API + TypeScript）
- ✅ Vue 2 支持（Options API）

### 📘 TypeScript 支持
- ✅ 完整的类型定义
- ✅ 类型安全的 API

### 🔧 优化
- ✅ 支持 ESM 和 CommonJS 模块格式
- ✅ 支持 Vite、Webpack 等构建工具
- ✅ CSS 样式独立提取
- ✅ Source Map 支持

### 🐛 已知问题
- ⚠️ 复杂 PDF 排版可能无法完美还原
- ⚠️ 图片水印去除功能有限
- ⚠️ OCR 识别率取决于图像质量
